About LeafAI

Banana Leaf Disease Detection using Deep Learning

Project Overview

LeafAI is a web-based disease detection system built specifically for banana plant health monitoring. It uses a convolutional neural network trained on real banana leaf images to identify common fungal diseases early, helping farmers take timely action to protect their crops.

Early detection of banana leaf diseases can reduce crop loss by up to 40%. This system brings that capability to any device with a camera and internet connection.

LeafAI Disease Detection System

Model Architecture
  • Base model: MobileNetV2 pretrained on ImageNet
  • Transfer learning with two-phase training
  • Phase 1: Custom classification head training
  • Phase 2: Fine-tuning top 30 base layers
  • Input size: 224 × 224 pixels
  • Output: 4-class softmax classifier
Tech Stack
  • Backend: Django 4.2
  • Deep Learning: TensorFlow / Keras
  • Model: MobileNetV2
  • Frontend: Bootstrap 5
  • Database: SQLite
  • Dataset: BananaLSD
